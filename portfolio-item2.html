Project: Online Cake Ordering App (Cherry Cakes)

Jul 2021 | Android Application Development

Cherry Cakes was my introduction to mobile application development. Designed during a time when mobile accessibility was booming, this project focuses on connecting local bakeries with customers through a seamless, digital storefront. It covers the entire journey from browsing flavor categories to secure checkout.

The Development Journey: Key Learnings

  • Android Fundamentals: I built this using Java and the Android SDK, which taught me about Activity lifecycles, Intent handling, and mobile UI/UX principles.
  • Backend-as-a-Service (BaaS): Integrating Firebase Realtime Database was a breakthrough moment, showing me how to handle real-time order updates and user authentication without building a custom backend from scratch.
  • User-Facing Design: I focused on creating a clean, appetizing interface that prioritizes product visualsâ€”a critical factor for any food-related application.
  • Mobile Payments Flow: While a simulation, implementing the logic for a secure payment and delivery tracking flow taught me about the importance of transactional integrity.

Technical Stack

  • Language: Java
  • Platform: Android SDK
  • Database: Firebase (Auth & Realtime Database)
  • Version Control: GitHub
