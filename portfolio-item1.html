Project: Excel Dashboard Analysis

Nov 2021 | Data Visualization Project

This project involved creating a comprehensive sales dashboard using Microsoft Excel. The primary goal was to transform a raw sales dataset into an interactive, visual tool for business analysis.

Key Objectives

  • Created Pivot Charts for key sales categories.
  • Implemented Spark lines for trend analysis in profits and sales.
  • Analyzed top customers and sales team performance.
  • Integrated Slicers for dynamic data filtering.
  • Established data models and relationships between multiple sheets.

Project Resources

  • GitHub Repository
  • View Full Report (Google Drive)

Technical Details

Modern data management often starts with Excel. This project demonstrates advanced usage of Excel's features, including complex formulas for profit calculation and geographical mapping of sales data.

The dashboard was designed to be user-friendly and aesthetically professional, avoiding clutter while highlighting the most impactful metrics. Challenges included selecting the right chart types for different data distributions and optimizing the filter chain.
