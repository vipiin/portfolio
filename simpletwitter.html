Project: Simple Twitter App

Social Media Concept | Vanilla Web Development

This project was my "back to basics" challenge. After working with heavy frameworks like Spring and Angular, I wanted to build something purely with the native features of the browser. It's a lightweight social media concept that handles state, persistence, and complex DOM interactions entirely through Vanilla JavaScript.

The "Vanilla" Experience: Key Learnings

  • DOM Manipulation: I learned how to efficiently update the UI without the overhead of a virtual DOM, gaining a deeper appreciation for how frameworks actually work under the hood.
  • Browser Storage: Implementing a tweet feed using LocalStorage was a great lesson in persistent client-side data management and its limitations.
  • Clean Architecture: Even without a framework, I focused on writing modular, readable JavaScript that separates data logic from UI rendering.

Core Features

  • Local Persistence: Stores tweets in the browser using LocalStorage.
  • Interactive UI: Real-time like and unlike functionality.
  • Clean Design: Focused on user-friendly interactions and responsiveness.

Technical Details

Built using standard **HTML5**, **CSS3**, and **Vanilla JavaScript**. The project focuses on efficient DOM manipulation and state management without heavy frameworks.
