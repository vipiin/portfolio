Project: Simple Twitter App

Social Media Concept | Web Development Project

This project is a lightweight, frontend-driven social media application that demonstrates core web interactions and local data persistence.

Core Features

  • Local Persistence: Stores tweets in the browser using LocalStorage.
  • Interactive UI: Real-time like and unlike functionality.
  • Clean Design: Focused on user-friendly interactions and responsiveness.

Technical Details

Built using standard **HTML5**, **CSS3**, and **Vanilla JavaScript**. The project focuses on efficient DOM manipulation and state management without heavy frameworks.
