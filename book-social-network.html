Full-Stack Web Application | Social Platform for Book Lovers

The Book Social Network is a full-stack platform designed to bridge the gap between keeping a personal reading log and engaging with a community. The Engineering Journey & Key Learnings

  • Stateless Security: Implementing JWT-based authentication gave me a deep understanding of how to secure a distributed application without relying on server-side sessions.
  • Cloud Service Integration: Learning to orchestrate multiple third-party APIs like Cloudinary (for images) and Brevo (for automated emails) was crucial for building a professional-grade product.
  • Angular Mastery: This project was where I truly grasped Angular's component lifecycle, reactive forms, and efficient state management using TypeScript.
  • Polyglot Deployment: Managing a backend on Koyeb, a database on Supabase, and a frontend on Vercel taught me the nuances of cross-origin configuration (CORS) and environment management.

Technical Stack & Deployment

  • Frontend: Angular with TypeScript, responsive UI design.
  • Backend: Spring Boot (Java) with RESTful API architecture.
  • Database: PostgreSQL (via Supabase).
  • Static Assets: Cloudinary for efficient image management.
  • Email Services: Brevo for integrated mail notifications.
  • Security: JWT (JSON Web Tokens) for secure stateless authentication.
  • Deployment: Hosted on Koyeb (Backend) and Vercel (Frontend).

Key Features
