Project: Book Social Network

Full-Stack Web Application | Social Platform for Book Lovers

The Book Social Network is a robust platform that allows users to manage their personal libraries, share book recommendations, and interact with a community of readers. It features a complete end-to-end implementation from a responsive frontend to a secure, scalable backend.

Live Application (Vercel) | GitHub Repository

Technical Stack & Deployment

  • Frontend: Angular with TypeScript, responsive UI design.
  • Backend: Spring Boot (Java) with RESTful API architecture.
  • Database: PostgreSQL (via Supabase).
  • Static Assets: Cloudinary for efficient image management.
  • Email Services: Brevo for integrated mail notifications.
  • Security: JWT (JSON Web Tokens) for secure stateless authentication.
  • Deployment: Hosted on Koyeb (Backend) and Vercel (Frontend).

Key Features

  • Secure Authentication: Built-in registration and account validation.
  • Library Management: Full CRUD operations for books in your personal collection.
  • Community Interaction: Real-time social features and book reviews.
  • Efficient Scaling: Leverages cloud services for media, data, and deployment.

This project demonstrates proficiency in modern full-stack development, integrating multiple professional cloud services to create a stable, production-ready application.
