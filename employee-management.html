Project: Employee Management System

Full-Stack Admin Portal | Java 17, Spring Boot, Angular 20

This project is a high-performance HR administration tool designed for real-time employee data management. It focuses on clean RESTful architecture and reactive frontend patterns to ensure a professional user experience.

The Engineering Journey & Key Learnings

  • Reactive State Management: Using Angular 20's reactive forms and signals (or standard observables) to handle complex input validation and real-time UI updates.
  • JPA/Hibernate Optimization: Implemented sophisticated entity mappings and repository patterns to manage database operations efficiently.
  • Zero-Infrastructure Database: Leveraged Neon Serverless PostgreSQL, which taught me about the advantages of cloud-native, auto-scaling databases.
  • Hybrid Cloud Deployment: Orchestrated the deployment of a Spring Boot backend on Render while serving the Angular frontend via GitHub Pages.

Technical Stack & Deployment

  • Frontend: Angular 20, TypeScript, TailwindCSS (for styling).
  • Backend: Java 17, Spring Boot, Spring Data JPA.
  • Database: PostgreSQL (Hosted on Neon).
  • Deployment: Render (Backend), GitHub Pages (Frontend).

Key Features

  • Full CRUD Operations: Comprehensive management of employee profiles.
  • Search & Filter: Dynamic filtering system for large employee datasets.
  • Responsive Dashboard: Optimized for both desktop and mobile administration.
  • Stateless API: Clean separation of concerns between client and server.
